# Project Context

## Domain Language

This is a language learning application for English exams (CET-4, CET-6, 考研, TEM).

## Key Concepts

- **Exam Papers**: PDF files of past exam papers
- **Sentences**: Key sentences extracted from exam papers
- **Vocabulary**: Words and phrases from exam papers
- **Annotations**: User notes and highlights

## Architecture

- **Frontend**: React + TypeScript + Vite
- **Backend**: Spring Boot (Java)
- **Storage**: Local files and in-memory storage
